GET EMAIL LIST

Used to retreieve emails from database.

URL : /api/email/

Method : GET

Auth required : NO

Success Response

Code : 200 OK

Content example

[
    {
        "_id": {
            "$oid": "59c64b8b5e751139e131d0d2"
        },
        "message": "Valid",
        "email": "user@mail.com",
        "code": 1
    },
    {
        "_id": {
            "$oid": "59c64dcc5e75113adac77c99"
        },
        "message": "Valid",
        "code": 1,
        "email": "user@mail.com",

    },
]


POST EMAIL

Used to post a email.

URL : /api/email/

Method : POST

Headers: Content-Type : application/json

Auth required : NO

Data constraints

{
    "email": "[valid email address]",
    "key": "[valid key]"
}

Data example

{
    "email":"user@mail.com",
    "key":"XXXXXXXXXXXXXXXXXXXXXXXX"
}

Success Response

Code : 200 OK

Content example

{
    "_id": {
        "$oid": "59c6acb65e751167aad1a179"
    },
    "message": "Valid",
    "code": 1,
    "email": "user@mail.com"
}

Error Response

Condition : If key not provided.

Code : 200 OK

Content :

{
    "response": "Unauthorized user"
}



Condition : If email and key missing

Code : 200 OK

Content :

{
    "response": "request cannot be empty!"
}



Condition : If no json data sent

Code : 400 BAD REQUEST

Content :

{
    "message": "Failed to decode JSON object: Expecting value: line 1 column 1 (char 0)"
}
